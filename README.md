# IT Lost and Found

Infrastructure, home labs, automation, and AI builds — by Michael Yates.

This is my corner of the web. A portfolio, a blog, and a record of things I've built, broken, and automated along the way.

The content spans what I've done professionally in security engineering and infrastructure, what I've proven in my home lab, and where I'm heading with AI and automation. Blog posts are written past tense — they document what happened, not what might. The About page ties it together.

Built with [Jekyll](https://jekyllrb.com/) and the [Minimal Mistakes](https://github.com/mmistakes/minimal-mistakes) theme, hosted on [GitHub Pages](https://pages.github.com/).

Deployed at [itlostandfound.github.io/Jekyll.GitHub](https://itlostandfound.github.io/Jekyll.GitHub/)
