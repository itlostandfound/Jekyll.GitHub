## F5 ASM — Application Security Manager

Web Application Firewall built into BIG-IP. Protects apps from OWASP Top 10 and custom attack signatures.

### Key Topics

- **Security Policies** — Automatic vs. manual, staging vs. enforcement
- **Signatures** — Attack signature sets, custom signatures, tuning
- **Positive Security** — Allowlists, URL/parameter/filetype restrictions
- **Violations** — Understanding what triggers, staging violations, learning
- **Blocking vs. Transparent** — When to block, when to observe
- **Policy Builder** — Traffic learning, suggestion review, deployment
- **Logging & Reporting** — Request logs, violation details, analytics
