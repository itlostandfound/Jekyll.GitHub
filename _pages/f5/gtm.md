## F5 GTM/DNS — Global Traffic Manager

DNS-based global server load balancing. Distributes traffic across data centers based on topology, availability, and policy.

### Key Topics

- **Wide IPs** — DNS names that resolve to the best pool member
- **Pools** — Round-robin, ratio, topology, LDNS-aware
- **Topology Records** — Geo-based routing, region matching
- **Monitors** — BIG-IP monitor vs. external check vs. virtual server status
- **iRules on DNS** — When the built-in options aren't enough
- **DNS Express** — Authoritative DNS zone hosting on BIG-IP
- **GTM vs. LTM** — Understanding where each fits
