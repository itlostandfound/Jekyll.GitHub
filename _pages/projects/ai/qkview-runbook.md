## QKView LLM Wiki Brain

Originally based on Andrej Karpathy's "LLM Wiki", but quickly pivoted to what Claude describes as a "runbook". Extracts a BIG-IP QKView and uses programmatic analysis scripts whose output is then agentically analyzed based on the runbook/Wiki Brain for a structure-constrained, professionally humanized output report.

Built-in self-improvement harness that allows the agentic analysis to alter its behavior with supplied reason and results in a deviation file per run that is then re-ingested into the runbook.
